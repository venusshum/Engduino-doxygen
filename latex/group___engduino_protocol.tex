\subsection{Detailed Description}
This is the driver code for the Engduino protocol.

The Engduino protocol enables communication between Engduino board and external devices such as personal computer or smart phone/tablet over the small set of instructions/commands. Commands are represented by easy to understand A\+S\+C\+I\+I strings with predefined definitions of different parameters. Commands can be send over the terminal which supports U\+A\+R\+T communication e.\+g. Putty, Hyper\+Terminal, Matlab, etc or over the Bluetooth communication. Bluetooth communication can be established between Engduino Bluetooth extension board and a Smart device (mobile phone, tablet, etc) or personal computer with Bluetooth port. In the case of Bluetooth communication Bluetooth extension board for Engduino need to be used.

The main sketch implementing this protocol\+: ..\textbackslash{}100.\+Engduino.ino

Engduino package\+: Start chr, delimiter, end chr \{ ; \}

Simplest Engduino package\+: \{P\+A\+C\+K\+A\+G\+E\+\_\+\+T\+Y\+P\+E; C\+O\+M\+M\+A\+N\+D\+\_\+\+I\+D; V\+A\+L1, ..., V\+A\+Ln\}

Examples\+: \{1;10;1\} or with defines \{P\+A\+C\+K\+A\+G\+E\+\_\+\+T\+Y\+P\+E\+\_\+1; C\+O\+M\+\_\+\+S\+E\+T\+\_\+\+L\+E\+D\+S; G\+R\+E\+E\+N\} -\/$>$ Turns on all R\+G\+B L\+E\+Ds with green colour.

\{1;100\} or with defines \{P\+A\+C\+K\+A\+G\+E\+\_\+\+T\+Y\+P\+E\+\_\+1; C\+O\+M\+\_\+\+G\+E\+T\+\_\+\+V\+E\+R\+S\+I\+O\+N\} -\/$>$ Returns package as \{1;100;V\+E\+R\+S\+I\+O\+N\}

\{1;111;100\} or with defines \{P\+A\+C\+K\+A\+G\+E\+\_\+\+T\+Y\+P\+E\+\_\+1; C\+O\+M\+\_\+\+G\+E\+T\+\_\+\+T\+E\+M\+P\+E\+R\+A\+T\+U\+R\+E; 100\} -\/$>$ Enables continuous mode on Temperature sensor. Package with measurement of temperature will be sends back on every 100 milliseconds. Sample\+: \{1;111;27546\} represents 27.\+546°\+C.
